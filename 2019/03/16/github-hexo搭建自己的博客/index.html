搭建自己的博客的苦逼经历

撰写于 2019-03-16 修改于 2019-03-26 分类 搭站美化 标签 hexo

本人才入安全“坑”的小白一枚，对小白来说入坑的前期主要事情就是百度，看大佬们发表的博客，特别看到有些大佬博客页面各种炫酷，心里那叫一个羡慕啊！终于在大佬w22的鼎立帮助下搭建了自己的博客。。。

苦逼的地方

1 当然都是因为自己是小白！！！

2 环境搭建好后，上传时发现没上传成功，找了会，最后发先竟然是因为repository打成别人的了。。。（当时心里一万只草泥马奔腾而过。。）

3 更换主题后，发现主题没改变，又是一顿cao百度，终于功夫不负小白心，原因时.git根目录下没README这个解析域名的文件。。。

忠告

当做一件事情，特别是自己从未做过的事情时，一定一定不要怕出错，因为只有出错了，才能更好的完善！

总之一句话啦，遇事不能怂，怂-是一败涂地的罪恶源头！哈哈~（有些装13）
